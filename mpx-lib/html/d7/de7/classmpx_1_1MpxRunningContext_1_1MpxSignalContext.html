Detailed Description

signal handler

class represents signal handler for signals delivered by OS. signal handling functionality of this class is based on sigaction mechanism. See 'man 7 signal' for additional info about sigaction.

create signal handler

function saves following informations in newly created instance of signal handler:

  • ctx - reference to instance of I/O multiplexer which created signal handler. It will be used by signal delivery to select correct I/O multiplexer when redirecting signals
  • signo - number of signal to be handled by this handler
  • f - call-back function to be used when handling signal
  • appdata - general reference to data provided by application
