MpxTaskMultiplexer * mpx::MpxEnvironment::_CreateTaskMultiplexer ( const char *  connStr,
bool  isWorker 
)
private

create new task multiplexer

this function creates new instance of MpxTaskMultiplexer class. its reference is saved into internal data structure m_tskmpxset, set of task multipexing objects. Access to this set is synchronized to prevent concurrent write operations on it. Every instance of task multiplexer is associated with unique pipe object used to properly terminate its execution. Execution termination is further described in description of MpxTaskMultiplexer class itself

Parameters
connStrconnection string parameter which is transfered to MpxTaskMultiplexer unchanged and it is not examined by this function $param isWorker working thread indicator transferred to MpxTaskMultiplexer constructor
